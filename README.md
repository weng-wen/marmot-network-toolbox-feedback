# Marmot Network Toolbox Feedback

Public issue tracker for [Marmot Network Toolbox](https://network.acgmole.top),
an offline-ready PWA for network planning and browser diagnostics.

土拨鼠的网络工具箱公开反馈区，用于提交问题、功能建议和浏览器兼容性报告。

## Before posting / 提交前

- Search existing issues first. / 请先搜索是否已有相同问题。
- Include browser, operating system, installed-PWA status and reproducible
  steps. / 请提供浏览器、系统、是否安装为 PWA 以及复现步骤。
- Screenshots are welcome after removing private information. / 截图请先移除隐私信息。
- Never post Wi-Fi passwords, Local Link pairing codes or SDP, private IP
  layouts, HAR secrets, certificates/private keys, or sensitive file content.
  / 请勿提交 Wi-Fi 密码、Local Link 配对码或 SDP、私有网络结构、HAR 密钥、
  证书私钥或敏感文件内容。

The toolbox has no account system. A GitHub account is required only to create
or reply to an issue. Tool inputs and results are not attached automatically.

工具箱本身没有账号系统；仅在创建或回复 Issue 时需要 GitHub 账号，工具输入和
结果不会被自动附加。
